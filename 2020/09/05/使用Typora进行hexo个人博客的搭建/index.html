在typora中编写hexo博客基础教程

  • 在windows命令行切换到管理员权限：runas /noprofile /user:Admin cmd

  • windows命令行中查看当前路径：chdir

  • 在windows中查看当前文件夹命令：dir

  • 初始化博客命令：sudo hexo init

    • 启动博客命令：hexo s 端口为：localhost:4000（起到实时预览作用，并未真正上传）
    • 新建文章命令：hexo n “文章标题”
      • 文章所在位置为source/_post
    • 生成文章命令为：hexo g
